# TG WS Proxy (Android)

[![Build Android APK](https://github.com/FunnyNosok/TGWSANDROID/actions/workflows/build.yml/badge.svg)](https://github.com/FunnyNosok/TGWSANDROID/actions/workflows/build.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Локальный MTProto-прокси сервер для Telegram на Android. 
Это приложение позволяет обходить блокировки Telegram, перенаправляя трафик через WebSocket-соединения и резервные маршруты Cloudflare.

Данный проект является **неофициальным портом и модификацией** оригинального десктопного приложения ПК. 

Оригинальный репозиторий: [Flowseal/tg-ws-proxy](https://github.com/Flowseal/tg-ws-proxy?tab=readme-ov-file)

---

## 📱 Особенности Android-версии

Мы взяли ядро оригинального прокси и адаптировали его для мобильных устройств, добавив ряд улучшений для более быстрой работы Telegram:

- **Фоновая работа:** Прокси работает как Foreground Service (непрерывная работа в фоне с уведомлением).
- **React Native UI:** Удобный мобильный интерфейс для управления настройками, генерации ссылок и просмотра статистики.
- **Улучшенный Cloudflare Fallback:** Использование Anycast IP-адресов Cloudflare вместо доменных имен.

## 🚀 Как это работает

Приложение поднимает локальный MTProto прокси (по умолчанию `127.0.0.1:1443`). 
Оно перехватывает трафик официального клиента Telegram и упаковывает его в зашифрованный WebSocket. 

Маршрутизация:
`Telegram Client` → `Локальный MTProto Proxy (127.0.0.1:1443)` → `WebSocket (TLS) / Cloudflare` → `Telegram DC`

## ⚙️ Использование

1. Установите и запустите приложение.
2. (Опционально) В настройках включите `CF Proxy Priority`.
3. Нажмите "Запустить прокси".
4. Нажмите "Открыть в Telegram" (или скопируйте ссылку и перейдите по ней) для автоматической настройки официального клиента.

## ⚠️ Решение проблем (Не грузит фото/видео)

Если в Telegram не загружаются медиафайлы (особенно актуально для аккаунтов без Telegram Premium):
- Зайдите в настройки прокси в приложении.
- В поле `DC -> IP` оставьте только `4:149.154.167.220`.
- Если не помогло, удалите все значения из этого поля.
- Вы также можете настроить свой собственный домен (CfProxy User Domain) через Cloudflare Workers, как описано в оригинальном репозитории.
